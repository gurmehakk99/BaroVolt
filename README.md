# BaroVolt
his is the Barovolt Simulator I built as my first project, which helps to simulate how different materials would react to voltage and temperature changes. The project is based on Peltier effect.

I've used Aluminium, Copper and Ceramics for materials which the user can choose, for which I included switch case. It calculates the cooling rate based on Voltage and then estimates the time needed to reach the target temperature.
